HTML Newsletter

Skills:  Code and Design


Mocha Moms, inc. is a non-profit organization. I coded the Mocha Moms Reston, Loudoun & Fairfax local chaper's monthly newsletter. The Mocha Moms of Reston newsletter contains announcements from the chapter president, upcoming events, parenting resources and food recipes that members can use while staying up to date. The newsletter is responsive using media queries.

The first set of images below is a sample of the sliced photoshop images used in the newsletter. They are in large, medium and small sizes to fit the sizes of device media including desktop, laptop and mobile views.

The second set of images below is a mockup of the newsletter produced in photoshop. I then coded the newsletter to represent the image including typography and colors specified in the mockup.
